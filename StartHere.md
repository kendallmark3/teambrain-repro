What We Demonstrated in the Demo

In the demo we showed a simple example of Intent-Driven Engineering using Claude Code:
	1.	Describe the intent of the system
	2.	Let Claude generate the application
	3.	Run the system locally
	4.	Validate the result in the browser

Example:
	•	React UI (Dice Roller)
	•	Node.js / TypeScript API
	•	Generated and executed through Claude Code

Goal:
Move from writing code manually → to generating a working system from intent.

“We didn’t just generate code — we generated a working system.”
