# AI Engineering Starter Repository

This repository demonstrates a modern **AI-assisted development workflow** using Claude Code.

The goal is to combine:

- human architecture guidance
- repository knowledge
- AI coding assistance

to accelerate software development.

---

## Quick Start

Install Claude Code CLI.

Then run:

```
claude
```

Inside Claude:

```
/init
```

This will analyze the repository and generate project context.

---

## Repository Structure

| Folder | Purpose |
|------|------|
| intent | High-level engineering intent |
| architecture | System design documentation |
| src | Application source code |
| devops | Deployment and pipeline configuration |
| agents | AI workflow definitions |

---

## Recommended Workflow

1. Read the intent files
2. Review architecture documentation
3. Use Claude to implement features
4. Validate with tests and CI pipelines
